## Agent skills

### Issue tracker

Issues are tracked as GitHub issues in this repo (`djp440/extensions`). See `docs/agents/issue-tracker.md`.

### Triage labels

The five canonical triage roles use their default names (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Multi-context — a `CONTEXT-MAP.md` at the root points to per-plugin `CONTEXT.md` files. See `docs/agents/domain.md`.
